# S3CmdBin

## Installation

Add this line to your application's Gemfile:

    gem 's3_cmd_bin'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install s3_cmd_bin

## Usage

    require 's3_cmd_bin'
    puts S3CmdBin::s3cmd
